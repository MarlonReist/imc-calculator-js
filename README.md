# Calculadora de IMC JS

Projeto de estudo desenvolvido com HTML, CSS e JavaScript, com foco em manipulação do DOM, eventos, validação de entradas e cálculo de IMC.

## Funcionalidades
- Cálculo de IMC
- Validação de campos
- Restrição de caracteres nos inputs
- Exibição da classificação do IMC
- Tabela de referência com classificações
- Botão para limpar campos
- Layout responsivo

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo do projeto
Praticar conceitos fundamentais de front-end, especialmente:
- seleção e manipulação de elementos no DOM
- eventos
- funções
- arrays de objetos
- renderização dinâmica de conteúdo
